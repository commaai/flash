# flash

â–º [flash.comma.ai](https://flash.comma.ai)

This tool allows you to flash AGNOS onto your comma device. Uses [qdl.js](https://github.com/commaai/qdl.js).

Built with **Svelte** for simplicity, brevity, and performance.

## Development

```bash
bun install
bun dev
```

Open [http://localhost:5174](http://localhost:5174) with your browser to see the result.

You can start editing the page by modifying `src/app/App.svelte`. The page auto-updates as you edit the file.

## Tech Stack

- **Svelte 5** - Reactive UI framework
- **Vite** - Fast build tool with instant HMR
- **TailwindCSS** - Utility-first CSS
- **WebUSB** - Direct device communication
